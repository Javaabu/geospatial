## Introduction

Adds Geospatial attribute support to Laravel projects

For Laravel 9 support use version 0.*

## Installation
You can install the package via composer:

```bash
composer require javaabu/geospatial
```

## Testing

You can run the tests with

``` bash
./vendor/bin/phpunit
```
